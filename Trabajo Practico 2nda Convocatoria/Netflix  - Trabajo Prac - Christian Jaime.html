Usuarios de Netflix¶

Autor: Christian Jaime¶

Fuente: https://www.kaggle.com/datasets/arnavsmayan/netflix-userbase-dataset¶

El set de datos seleccionado es una muestra de la base de usuarios de Netflix y contiene el tipo de suscripción, género, edad, país, fecha de la suscripción, etc.

Con el mismo se intentará encontrar patrones de los sucriptores, tales como algún género, edad o nacionalidad dominante en la plataforma.

Otro aspecto interesante será poder ver cuánto tiempo llevan en la plataforma los usuarios, lo que dará una idea del éxito de la misma.

Antigüedad¶

Aunque no tenemos la antigüedad como parte de los datos, podemos usar la fecha de inscripción y la fecha del último pago de los usuarios para calcularla.

In [216]:
data['Last Payment Date'] = pd.to_datetime(data['Last Payment Date'])
data['Join Date'] = pd.to_datetime(data['Join Date'])

data['Antiguedad'] = ((data['Last Payment Date'] - data['Join Date']).dt.days)/30/12

g = sns.displot(data=data, x="Antiguedad", height=6, aspect=2,)
g.set_axis_labels("Antigüedad","Usuarios")
plt.grid()
plt.show()

La información más interesante hasta ahora, viene de la distribución de la antigüedad. Del gráfico podemos observar que la lealtad a la plataforma dura como máximo 2 años, algo decepcionante para un servicio que inició su servicio de streaming desde 2008. Sin embargo, un detalle a considerar, es que existen usuarios con antigüedad negativa, lo que refuerza la idea de que el dataset sólo es de prueba.

import csv
import json

def convert_csv_to_dict(csv_file_path, json_file_path):
    data = []
    with open(csv_file_path, 'r') as csv_file:
        csv_reader = csv.DictReader(csv_file)
        for row in csv_reader:
            data.append(row)

    with open(js_file_path, 'w') as js_file:
        js_file.write("data =" + json.dumps(data, indent=4))

    print("CSV to DICT conversion complete.")

#Función para generar los datos usados en el gráfico Sunburst con D3

import csv
import json

def convert_row_to_string(row, column_order):
    values = []
    for column in column_order:
        value = row.get(column, "")
        values.append(str(value))
    return '_'.join(values)

def convert_csv_to_js(csv_file_path, js_file_path, column_order):
    data = []
    with open(csv_file_path, 'r') as csv_file:
        csv_reader = csv.DictReader(csv_file)
        for row in csv_reader:
            converted_row = convert_row_to_string(row, column_order)
            data.append(converted_row)

    with open(js_file_path, 'w') as js_file:
        js_file.write("data =" + json.dumps(data, indent=4))

    print("CSV to JS conversion complete.")

# Example usage
csv_file_path = 'Netflix Userbase.csv'
js_file_path = 'sunburst/Netflix_Userbase_sunburst.js'
column_order = ['Country','Gender','Subscription Type','Device',]

convert_csv_to_js(csv_file_path, js_file_path, column_order)

CSV to JS conversion complete.
